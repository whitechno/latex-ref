\begin{theorem}
If an operator has both Left Identity and
Right Identity then it is \emph{UNIQUE}.
\end{theorem}

\begin{proof}
Let $e_{l}$ be left identity 
$\therefore e_{l} * e_{r} \implies e_{r}$
\end{proof}
